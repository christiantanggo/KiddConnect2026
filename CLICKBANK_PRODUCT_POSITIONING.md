# ClickBank Product Positioning Guide

## Issue
ClickBank initially rejected the product because it was categorized as "phone services," which is prohibited.

## Solution: Reframe as AI Software
**Position Tavari as: AI Receptionist Software, NOT a telephony service**

The product being sold is **software** (AI receptionist software), not a phone service. Telnyx is just the underlying infrastructure (like AWS is for web apps), but the **product** is the AI software.

## Key Positioning Points

### ✅ What Tavari IS (Allowed on ClickBank):
- **AI Receptionist Software** - Digital software product
- **AI Customer Service Software** - Business software solution
- **Automated Business Assistant Software** - SaaS application
- **Digital Product** - Software delivered via web dashboard
- **Business Automation Software** - Tools for small businesses

### ❌ What to AVOID saying:
- "Phone service" or "Phone minutes"
- "Telephony service"
- "SMS service" or "Text messaging service"
- Anything that sounds like you're selling phone minutes or telephony

### ✅ What to EMPHASIZE:
- **Software product** - AI receptionist software
- **Digital delivery** - Access via web dashboard
- **Business automation tool** - Software for small businesses
- **AI assistant** - Software that handles customer inquiries
- **SaaS subscription** - Software as a Service

## Product Description for ClickBank

### Recommended Title:
**"Tavari AI: AI Receptionist Software for Small Businesses"**

### Recommended Description:
```
Tavari AI is AI receptionist software that helps small businesses automate customer communication. This digital software product includes:

- AI-powered receptionist software that handles customer inquiries
- Web-based dashboard for configuration and management
- Automated customer communication software
- Business automation tools for FAQs and messaging

This is a software subscription delivered via secure web dashboard. The software uses AI technology to assist with customer communication tasks.

Product Type: Recurring Digital Product (Software Subscription)
Category: Software / Business Tools
```
