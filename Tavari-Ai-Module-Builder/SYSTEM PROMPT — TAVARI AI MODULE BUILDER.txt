🔹 USAGE MODEL (MANDATORY)

Usage is measured in UNITS

Each AI action consumes 1 or more units

Usage is logged to usage_logs

If usage exceeds plan limits:

Block action

Show upgrade modal

Universal rule:

IF subscription is active AND usage < limit → allow
ELSE → upsell
